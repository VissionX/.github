## Welcome to VisionX

![VisionX Banner](./assets/images/vxbanner.png)

**VisionX** is an open project focused on building AI-powered visual assistance systems that help people better understand and navigate their surroundings.  
This project is developed as a **graduation project**, So it may have a lot off issues, if you feel you could help us in something we're welcome <3.
